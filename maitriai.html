During my Full Stack Developer Internship at MaitriAI, I developed a Biometric OCR System that automated identity verification by extracting and processing textual and biometric data from Aadhar and PAN cards. This project aimed to enhance document recognition accuracy, reduce manual verification efforts, and ensure secure identity authentication using a combination of OCR, deep learning, and database integration.

Project Technical Overview

The system was designed to handle Indian government-issued identity documents, particularly Aadhar cards (used for biometric-based identification) and PAN cards (used for financial and tax-related identification). The key challenge was to ensure high OCR accuracy despite variations in document quality, font styles, distortions, and handwritten elements.
Using EasyOCR and Pytesseract, an OCR engine powered by Tesseract, I implemented a robust text recognition pipeline capable of handling multiple fonts, languages, and distorted document scans. To enhance accuracy, I incorporated image pre-processing techniques such as noise reduction, thresholding, and contour detection, ensuring clear text extraction even from low-quality inputs.
A crucial component of this system was biometric data processing, where the extracted information was cross-verified against a secure SQL database.

Text Extraction

  • The Aadhar card OCR module was designed to extract Aadhar Number (12-digit unique identity number), Name of Cardholder, Date of Birth, Gender and Address.
  • The PAN card OCR module focused on extracting the Permanent Account Number (PAN) (10-character alphanumeric ID), Name of the Cardholder, Father’s Name, Date of Birth
    • The process involves preprocessing Aadhar and PAN card images using grayscale conversion, noise reduction, and skew correction, followed by OCR text extraction, QR code decoding, and database integration for verification and validation.

    Key Features

  • OCR-based Text Extraction: Implemented using Pytesseract, capable of recognizing structured and unstructured text from various document formats.
  • Deep Learning for Document Classification: Trained a Convolutional Neural Network (CNN) to classify document types, improving recognition accuracy.
  • Image Preprocessing: Used OpenCV to enhance document clarity through techniques like grayscale conversion, adaptive thresholding, and morphological operations.
  • Database Management: Developed a MySQL-based document storage system for efficiently storing, retrieving, and comparing extracted biometric details.

    • Performance Optimization

    Database query efficiency was enhanced through composite indexes on frequently joined columns like order_items(orderId, itemId), query caching for static menu data using Redis, and pagination implementation for order history displays. The search functionality employs MySQL full-text indexing on menu item names and descriptions, achieving average query response times under 120ms during load testing with 500 concurrent users.

    Challenges and Solutions

    One of the main challenges was dealing with noisy or skewed documents, which often led to incorrect text extraction. To solve this, I integrated automated image correction algorithms, including Hough Transform for line detection and perspective transformation, ensuring properly aligned inputs. Additionally, differentiating between handwritten and printed text posed a challenge. By leveraging TensorFlow-based character recognition models, I improved the system’s ability to process diverse text formats.
    Database efficiency was another crucial factor. To speed up queries, I indexed biometric fields and optimized search algorithms, significantly reducing response time during authentication checks.

    Impact and Aplications

    This Aadhar and PAN OCR system significantly streamlined the identity verification process, making it faster, more reliable, and automated. The system was particularly useful for:

    • Banking and Financial Sectors for KYC (Know Your Customer) compliance.
    • E-Governance Portals to validate identity documents for government schemes.
    • Employment and Background Verification for secure onboarding processes.

    Through this project, I deepened my understanding of OCR-based document recognition, deep learning for text extraction, and secure database management, making it a real-world application of AI-driven automation in identity verification.
