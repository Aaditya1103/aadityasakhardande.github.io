Hand Gesture Driven
Mouse System

GitHub

The Hand Gesture Driven Mouse System is an innovative project designed to replace traditional mouse input with natural hand gestures, enabling intuitive and seamless interaction with computers. This system integrates state-of-the-art computer vision and gesture recognition technologies to track hand movements in real time and convert them into mouse actions such as moving the cursor, clicking, and taking screenshots.

Features

  • Real-Time Fingertip Tracking: The system captures hand movements using a webcam and tracks the position of the index fingertip with high precision. This allows for smooth and responsive cursor control.
  • Gesture-Based Actions: Specific gestures are mapped to mouse actions. For example, bringing the thumb and index finger close together simulates a left-click, while other gestures enable right-click, double-click, or taking screenshots.
  • Dynamic Scaling: The cursor's movement is scaled dynamically based on screen dimensions to ensure accurate positioning.
  • One-Handed Operation: The system is designed for single-hand use, simplifying the user experience.

Technologies Used

  • OpenCV: For processing real-time video frames and detecting hand gestures.
  • MediaPipe: To identify and track hand landmarks, ensuring accurate detection of fingertip positions.
  • PyAutoGUI: To control the mouse cursor, simulate clicks, and take screenshots.
  • Pynput: For low-level mouse control, enabling precise actions like left and right clicks.

Workflow and Functionality

  • Video Capture: The system uses a webcam to capture live video, which is processed frame by frame.
  • Hand Landmark Detection: MediaPipe is used to identify 21 key landmarks on the hand. The positions of these landmarks are analyzed to determine gestures.
  • Gesture Recognition: Gestures are detected based on the relative positions of the hand landmarks. For example, the distance between the thumb and index finger determines certain actions.
  • Mouse Control: PyAutoGUI translates the detected gestures into mouse movements and actions, such as moving the cursor, clicking, or taking screenshots.
  • Dynamic Feedback: The system provides real-time feedback by overlaying visual indicators on the video feed, displaying the detected gesture or action.

Challenges Faced

  • Lighting Variations: Ensuring accurate hand landmark detection under varying lighting conditions required robust preprocessing techniques.
  • Latency: Minimizing the delay between gesture detection and cursor movement to achieve real-time responsiveness.
  • Gesture Ambiguity: Differentiating between similar gestures to prevent unintended actions.
  • Hardware Compatibility: Adapting the system to work seamlessly across different webcams and screen resolutions.

Impact

The Hand Gesture Driven Mouse System offers an innovative approach to human-computer interaction, providing a touchless and accessible alternative to traditional input devices. It is particularly beneficial for individuals with physical disabilities, offering a more inclusive way to interact with technology. Additionally, the system has applications in scenarios requiring hands-free operation, such as medical environments or industrial settings.
