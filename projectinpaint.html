LiveInpaint: Real-Time Object Removal
and Video Restoration

This project implements a deep learning-based image inpainting system to remove unwanted objects from video frames using Generative Adversarial Networks (GANs). The model reconstructs occluded regions with semantically coherent content while maintaining spatial and contextual consistency.

Features

  • Frame-Wise GAN Inpainting: Each frame is processed individually using a Partial Convolution-based GAN architecture.
  • Mask Generation: Automatic mask creation using contour detection and manual annotations for user-defined regions.
  • Temporal Smoothness: Maintains coherence across frames using optical flow-guided mask warping.
  • Batch Processing: Processes video frames at 24 FPS, averaging 0.21s per frame on a GTX 1660 Ti.

Technologies Used

  • PyTorch: For implementing the GAN architecture (generator + discriminator).
  • OpenCV: Used for video frame extraction, mask creation, and post-processing.
  • FFmpeg: For frame-to-video recombination post inpainting.
  • NumPy / PIL: For data preprocessing and format conversions.

Technical Workflow

  1. Input: Accepts .mp4 video or raw frame sequence.
  2. Preprocessing: Each frame is resized to 256Ã—256, masks are applied to the ROI.
  3. GAN Inference: Generator fills masked regions while discriminator ensures realism.
  4. Post-Processing: Edges are blended with Laplacian filters for seamless output.
  5. Output: Video reconstructed using inpainted frames at original FPS.

Performance Evaluation

  • PSNR: 32.5 dB (mean) across test frames with known ground truth.
  • SSIM: 0.89 average structural similarity index.
  • Artifact Rate: < 3% in corner regions after edge-blending pass.

Challenges & Solutions

  • Artifact Reduction: Introduced Total Variation Loss to reduce blurry fills and edges.
  • Generalization: Trained on Places2 dataset for broader environmental adaptability.
  • Color Drift: Histogram matching applied during post-processing for color correction.
  • Frame Sync: Used DeepFlow-based optical flow to align dynamic objects over time.

Impact

This project demonstrates how GANs can be leveraged to enhance video post-production, enable privacy-preserving content anonymization, and restore archival footage. It opens pathways for media professionals, researchers, and AI applications requiring scene reconstruction without manual editing.
