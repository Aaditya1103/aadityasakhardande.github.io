Robotic Arm Control Motion Planning

GitHub

This project focuses on the modeling, control, and motion planning of robotic manipulators with increasing degrees of freedom. It begins with an analytical inverse kinematics solution for a planar two-joint arm, extends to Jacobian-based differential control for an N-joint planar manipulator, and culminates in sampling-based RRT* motion planning for a seven-joint spatial robotic arm with obstacle avoidance.

Robotic manipulators require precise coordination between kinematic modeling, control laws, and motion planning algorithms. This project explores three fundamentally different paradigms of arm motion generation: closed-form inverse kinematics, Jacobian-based differential control, and probabilistic sampling-based planning in high-dimensional configuration spaces. All systems are implemented in Python with real-time visualization.

Abstract

This project presents a progressive study of robotic arm control and motion planning across increasing degrees of freedom. A planar two-degree-of-freedom (2-DOF) arm is first controlled using analytical inverse kinematics with proportional feedback to ensure smooth convergence to arbitrary Cartesian targets. The approach is then generalized to an N-DOF planar arm using Jacobian-based inverse kinematics with damped least-squares inversion to address redundancy and singularity issues. Finally, a seven-degree-of-freedom (7-DOF) spatial manipulator is controlled using the RRT* (Rapidly-exploring Random Tree Star) algorithm in joint space, enabling collision-free path planning in environments with obstacles. The results demonstrate the strengths and limitations of local control versus global planning methods for robotic manipulators.

Introduction

Robotic arm manipulation is a core problem in robotics, spanning applications from industrial automation to surgical robotics and humanoid manipulation. While low-degree-of-freedom arms can often be controlled using analytical inverse kinematics, higher-degree-of-freedom and redundant systems require numerical and optimization-based approaches.

This project investigates how arm control strategies evolve with increasing complexity. Analytical inverse kinematics provides exact solutions but scales poorly. Jacobian-based methods enable smooth, local task-space control for redundant manipulators. Sampling-based planners such as RRT* operate in joint space and provide global solutions that respect kinematic constraints and obstacle avoidance.

System Architecture

The overall system is divided into three primary subsystems corresponding to different levels of abstraction in robotic manipulation:

  • Kinematic Modeling: Forward kinematics using serial-chain formulations and Denavit–Hartenberg parameters.
  • Control: Analytical and Jacobian-based inverse kinematics for real-time end-effector regulation.
  • Motion Planning: Sampling-based RRT* planning in joint space with collision checking.

Methodology

Analytical Inverse Kinematics (2-DOF Arm)

For the planar two-joint arm, closed-form inverse kinematics is derived using trigonometric relationships and the law of cosines. Given a desired Cartesian end-effector position (x, y), joint angles θ₁ and θ₂ are computed exactly, provided the target lies within the reachable workspace. To improve numerical stability and visualization smoothness, a proportional feedback controller is applied in joint space rather than directly commanding the solution.

When the target lies outside the arm’s reachable workspace, the solution is clamped to the workspace boundary, ensuring continuous and physically valid motion.

Jacobian-Based Inverse Kinematics (N-DOF Arm)

For the N-joint planar arm, inverse kinematics is solved using a differential formulation. The manipulator Jacobian maps joint velocities to end-effector velocities, allowing joint updates to be computed from Cartesian position error.

To handle redundancy and avoid numerical instability near singular configurations, a damped least-squares (DLS) pseudo-inverse is used:

θ̇ = Jᵀ (J Jᵀ + λ² I)⁻¹ e

This formulation ensures smooth convergence even in configurations where the Jacobian becomes ill-conditioned. Joint angles are updated incrementally using Euler integration, enabling real-time interactive control of the end effector.

RRT* Motion Planning (7-DOF Arm)

For the seven-degree-of-freedom spatial manipulator, motion planning is performed in joint space using the RRT* algorithm. Each node in the search tree represents a full joint configuration, and edges correspond to interpolated joint-space motions.

Collision checking is performed by computing the forward kinematics of the arm for each interpolated configuration and testing all link points against spherical obstacles in Cartesian space. RRT* improves upon standard RRT by rewiring the tree to minimize path cost, producing asymptotically optimal solutions.

Testing and Results

The analytical inverse kinematics controller demonstrated fast and stable convergence for all reachable targets, with smooth trajectories and predictable behavior. The Jacobian-based controller successfully handled redundancy and allowed flexible arm postures, though it exhibited slower convergence near singularities.

The RRT* planner consistently generated collision-free joint-space paths for the 7-DOF arm in cluttered environments. While computationally more expensive, it enabled global motion planning that local inverse kinematics methods cannot provide.

Conclusion

This project highlights the hierarchical nature of robotic manipulation. Analytical and Jacobian-based inverse kinematics are effective for local control and real-time interaction, while sampling-based planners such as RRT* are essential for global navigation and obstacle avoidance in high-dimensional systems. Together, these methods form the foundation of modern robotic arm autonomy.

Challenges

  • Jacobian singularities can degrade convergence speed.
  • RRT* planning becomes computationally expensive in high-dimensional spaces.
  • Local IK methods cannot guarantee collision-free motion.
